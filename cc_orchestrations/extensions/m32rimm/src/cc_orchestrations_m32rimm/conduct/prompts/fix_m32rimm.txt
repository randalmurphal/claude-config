## m32rimm-Specific Fixes

### Missing retry_run
```python
# BEFORE
db.collection.find_one({"id": id})

# AFTER
from fisio.common.mongo_helpers import retry_run
retry_run(db.collection.find_one, {"id": id})
```

### Missing subID filter
```python

db.businessObjects.find({"md.type": "assets"})

# AFTER

    "md.type": "assets",

})
```

### Missing flush before aggregation
```python
# BEFORE
db_ops.upsert_bo(bo, fingerprint=fp)
mark_for_aggregation(db, sub_id)

# AFTER
db_ops.upsert_bo(bo, fingerprint=fp)
db_ops.flush()
mark_for_aggregation(db, sub_id)
```

### Unpaired import tracking
```python
# BEFORE
insert_data_importer(db, scanner_id, ...)
process_data()
# missing complete_data_importer

# AFTER
insert_data_importer(db, scanner_id, ...)
try:
    process_data()
    complete_data_importer(db, scanner_id, stats, status='Finished')
except Exception as e:
    complete_data_importer(db, scanner_id, {}, status='Failed', error=str(e))
    raise
```

```python

# AFTER
```

```python

```
