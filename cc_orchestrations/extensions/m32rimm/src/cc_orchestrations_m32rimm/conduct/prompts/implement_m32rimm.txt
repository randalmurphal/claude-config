### MongoDB Operations

### Import Framework
```python
# ALWAYS pair import tracking
insert_data_importer(db, scanner_id, ...)
try:
    process_data()
    complete_data_importer(db, scanner_id, stats, status='Finished')
except Exception as e:
    complete_data_importer(db, scanner_id, {}, status='Failed', error=str(e))
    raise  # Re-raise after tracking
```

### Try/Except
```python
# ONLY wrap network/DB operations
try:
    response = requests.get(url)
except requests.RequestException as e:
    LOG.error("API call failed", extra={"url": url, "error": str(e)})
    raise

# NEVER wrap safe operations
data.get("key")  # Returns None, no try needed
items[0] if items else None  # Guard clause, no try
```
